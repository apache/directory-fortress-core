    • Build Apache Fortress Realm

    This section describes how to install The Fortress Realm Java EE Policy Enforcement Component

    1. Set java and maven home env variables.

    2. Change directory: 
                      
      
cd directory-fortress-realm/
    3. Build fortress realm with maven and install to local repo. The fortress rest [Build Apache Fortress Rest] and fortress web [Build Apache Fortress Web] are both dependent on the fortress realm. 
                          
      
mvn clean install
